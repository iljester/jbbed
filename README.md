### jbbed Code
BBCode Editor

### Description
A simple BB editor in JQuery fully customizable.

### Require
This plugin requires JQuery

### Basic Usage

```js
$('#mytextarea').jbbed();
```
Or if you want multiple instances in a single page, the advice is to use jbbedMu:

```js
$('.mytextarea').jbbedMu();
```

### Basic output in php

```php
echo jbbed_convert( $string );
```
